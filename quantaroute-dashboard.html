Architecture

From address list to Google Maps in seconds

A three-stage pipeline: geocoding → road-distance matrix → quantum optimisation. Each stage is modular, independently tested, and composable.

A
Stage 1 — Geocoder
Each address is cleaned and passed to the Nominatim geocoder. Lat/lng coordinates are returned, invalid addresses are flagged early and safely dropped from the route.
geocoder.py · Nominatim
B
Stage 2 — Road Matrix
An N×N distance matrix is built using OSRM (Open Source Routing Machine). Real road distances — not straight-line — between every pair of stops.
road_matrix.py · OSRM
C
Stage 3 — QAOA Solver
The distance matrix is fed into a QAOA circuit. Three candidates are evaluated: input order, nearest-neighbour greedy, and quantum-optimised. The shortest wins and is returned.
qaoa.py · Qiskit
D
Stage 4 — Route Builder
The winning stop order is combined with the start address and optional return-to-start flag to build the final Google Maps directions URL and WhatsApp message.
route_builder.py
E
Stage 5 — FastAPI backend
main.py exposes a clean API that accepts the address list, start address, and return flag. Benchmark fields are saved to SQLite route history for later analysis and case studies.
FastAPI · SQLite · Pydantic
F
Stage 6 — Frontend
A single-page HTML/CSS/JS frontend handles address input, start/depot entry, return toggle, and result rendering including benchmark details. Hosted on Render.com.
Vanilla HTML · Render.com

What powers QuantaRoute

A lean, modern Python stack — no unnecessary dependencies. Each layer is independently tested, file-by-file developed using a 100% AI-agent workflow.

Optimisation
Qiskit + QAOA
Quantum Approximate Optimisation Algorithm solving the Travelling Salesman Problem variant for multi-stop routing.
API backend
FastAPI
High-performance Python API. Pydantic models for clean request/response validation. Async-ready, auto-documented endpoints.
Geocoding
Nominatim / OSM
OpenStreetMap Nominatim converts raw address strings to lat/lng coordinates. No Google Maps API cost for geocoding.
Road distances
OSRM
Open Source Routing Machine builds the real road-distance matrix between all stop pairs. Real driving distances, not straight-line.
Data persistence
SQLite
Lightweight embedded database stores route history and benchmark metrics — original distance, nearest-neighbour, final selected, fuel saving %.
